Программа RadioGraph

Программа, созданная для построения сети тракинговой связи. Алгоритм обеспечивает частичную автоматизацию процесса разработки сети, строя граф с некоторыми ограничениями - длина пути от базовой точки не превышает 4, и из каждого узла выходит не больше трех веток.

Как работать с программой:

  • Для построения сети разместите на карте необходимые точки и нажмите кнопку "Построить"
  • Для редактирования связей с данной точкой нажмите на неё и в появившейся таблице настройте связи точек
  • Для удаления точки из сети нажмите в таблице "Удалить точку"
  • Для очистки сети нажмите "Очистить"
